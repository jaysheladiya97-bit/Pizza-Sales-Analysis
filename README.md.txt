# Pizza Sales Analysis Project

## Tools Used
- SQL (Data Analysis & KPIs)
- Excel (Dashboard & Reporting)

## Project Description
This project analyzes pizza sales data to identify key business insights such as:
- Total Revenue
- Average Order Value
- Best & Worst Selling Pizzas
- Sales Trends by Day, Hour, Category, and Size

## Files Included
- pizza_sales data_set.csv → Raw dataset
- SQLQuery1.sql → SQL queries used for analysis
- PIZZA SALES SQL QUERIES.docx → Query explanations
- Final.xlsx → Excel dashboard
